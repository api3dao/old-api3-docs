# {{$frontmatter.title}}

<VersionWarning/>

- Do not place double quotes (") around the value of each variable.
- Variable names cannot contain dashes (-) or start with a number.

```sh
AWS_ACCESS_KEY_ID=AKIA...GYZWRAHOH
AWS_SECRET_ACCESS_KEY=B87...UD/LEl6Z/ZCMXO9
```
